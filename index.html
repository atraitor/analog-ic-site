模拟 IC 模块总览：ADC / LDO / PLL

这里主要整理几个常见的模拟模块：ADC、LDO、PLL 等的基础概念、典型架构和学习要点，方便你按模块查阅和扩展自己的笔记。

ADC 模块

模数转换器，负责把模拟信号变成数字信号。关注分辨率、采样速率、INL/DNL、SNR、ENOB 等指标。

典型架构：SAR、Pipeline、Sigma-Delta 等。

LDO 模块

低压差线性稳压器，用于本地稳压、降压。关注压差、负载能力、PSRR、稳定性、静态电流等。

典型架构：带误差放大器 + pass 元件 + 参考。

PLL 模块

锁相环，用于时钟生成与频率合成。关注锁定时间、抖动、相噪、环路稳定性等。

典型结构：PD/CP + LPF + VCO + 分频器。
